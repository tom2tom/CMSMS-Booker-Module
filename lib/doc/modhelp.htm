What Does It Do?

It manages and displays bookings for resources, singly and/or in groups.

Any resource may be in one or more group(s). Any group may be in other group(s), creating a hierarchy.

Any group may be assigned to an authorized admin user.

How Do I Use It?

Start by creating resource(s), and group(s) if wanted, via the Content menu.

Place a smarty tag {cms_module module='Booker'} in a suitable page or template.

Requirements

  • PHP 5.3.6+
  • PHP MultiByte String extension
  • PHP Openssl extension
  • Until PHP ges its act together, its deprecated Mcrypt extension is needed for password hashing
  • CMSMS Authenticator module

Works better if you have

  • PHP Collator class
  • At least one of these PHP cache modules: apc[u],[p]redis,wincache,memcache,xcache,yac
  • CMSMS Notifier module
